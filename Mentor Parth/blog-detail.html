Cloud Computing
January 20, 2023
What is Cloud Computing?

Basically, in simple words, Cloud Computing is on delivery of IT resources over the internet.

Who uses Cloud Computing?

There is a high probability if you used a mobile phone or laptop you might have used cloud computing. Yes seriously if you used Gmail, Google Drive, Microsoft one drive, Google docs, stored files, played games, or used online services, etc all this comes under Cloud Computing services.

Why use Cloud Computing?
  • It is easy to use anyone can use Cloud Computing services
  • If one uses Cloud Computing one can save lots and lots of money in terms of maintenance, tech department, recovery, and all that stuff Cloud Computing helps to reduce the cost of overall services.
  • Easy to access you can access Cloud Computing services all over the world if you have internet connectivity.
  • Budget-friendly if you are a beginner with a startup or any business you can go with Cloud Computing because it seriously helps to make your budget-friendly services.
  • Its speed is very fast one can enjoy working on cloud computing services.
  • It is secured so one can freely use cloud computing services.
  • Backup is made very easy that's why Cloud Computing services are also known as reliable services.
Disadvantages of Cloud Computing?
  • The first and major concern in the digital world is, was, and always will be data security and privacy threats for common users.
  • Lack of control as a user we don't have that much control over Cloud Computing services.
  • It is very difficult to migrate from one Cloud Computing service to other.
  • Internet dependency yeah if you want to access cloud computing services it is a must that one has internet connectivity because cloud computing or you can say the digital world is dependent on the internet so make sure that you have good internet to use cloud computing service very smoothly.
Cloud Computing Services?
  • IaaS: IaaS stands for infrastructure as a service
  • Paas: PaaS stands for the platform as a service
  • SaaS: SaaS stands for software as a service

Examples of Cloud Computing.

There are many cloud computing services but some of the famous Cloud Computing examples are listed below if you want you can check out them.

  • Salesforce
  • Digital Ocean
  • AWS
  • Dropbox
  • Azure
  • Cybersecurity
